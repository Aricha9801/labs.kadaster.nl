Kadaster Data Science

Over Kadaster Labs

Op deze website verzamelen we de resultaten van data-gedreven experimenten die vanuit het Kadaster zijn uitgevoerd, vaak in samen werking met Kadaster partners. Deze experimenten en use cases maken veelal gebruik van Data Stories, browsers en/of demonstrators; deze zijn als losse componenten ook benaderbaar via de menu balk. De inhoud van deze website is experimenteel van aard en is bedoeld om te inspireren. Onder het kopje Dissemination geven we meer achtergrond over Data Science bij het Kadaster, en onder Data zijn de in de experimenten gebruikte datasets toegankelijk gemaakt.

Use Case
Energie Labels en Machine Learning
Voor het Geo Gebruikersfestival 2018 ontwikkelde het Kadaster een demo-applicatie voor het voorspellen van energielabels van panden, op basis van slechts een bouwjaar en een pandoppervlak. Het doel van de demo is aantonen dat met een eenvoudig machine learning model een support system kan worden gerealiseerd.
