Data Stories

Data stories are HTML document that are interpspersed with SPARQL queries and their corresponsing result set visualizations. Whenever a Data Story is loaded, the SPARQL queries return up-to-date results that are displayed on the fly. This makes Data Stories fully reproducible and grounded in the underlying data.

PDOK Knowledge Graph
Het PDOK platform (Publieke Data op de Kaart) bevat een grote hoeveelheid open datasets met een geo component. In PDOK Knowledge Graph bevragen we combinaties van datasets in PDOK, waardoor nieuwe inzichten worden opgedaan.
